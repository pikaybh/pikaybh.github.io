# How to

## local web

### debugging

```
$ bundle exec serve jekyll --drafts
```

it'll run on `http://127.0.0.1:4000/`.

you can change the port with `--port`.

`--draft` is for draft, which isn't ready to be published, posts.

### bundle error

Mostly it's because of the absent of bundle installing.
Use `$ bundle install`.

### webrick error

Try `$ bundle add webrick`.
